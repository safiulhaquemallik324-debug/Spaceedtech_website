SpaceEd Tech is a futuristic and responsive technology website designed to deliver a modern user experience. Built using HTML, SCSS, and Bootstrap, the project focuses on clean layouts, smooth responsiveness, reusable components, and an engaging visual design. The website showcases a professional tech-oriented interface with modern styling and optimized user interactions across desktop and mobile devices.

Tech Stack:

HTML5
SCSS (Sass)
Bootstrap 5
Responsive Design
Modern UI/UX Principles

Features:

Futuristic Design
Fully Responsive Layout
Interactive 3D Flip Cards
Clean and Structured SCSS Architecture
Mobile & Tablet Optimized
Bootstrap Components Integration
